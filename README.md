# Prática Avaliada 4

Projeto desenvolvido como atividade prática do curso Generation.

## 📚 Sobre o projeto

Este repositório contém exercícios de SQL envolvendo a criação e manipulação de bancos de dados, incluindo:

- Criação de tabelas;
- Inserção de registros (`INSERT INTO`);
- Consultas com `SELECT`;
- Relacionamentos entre tabelas utilizando `INNER JOIN`;
- Renomeação de colunas com `AS`;
- Ordenação de resultados com `ORDER BY`.

## 🛠️ Tecnologias utilizadas

- SQL
- MySQL
